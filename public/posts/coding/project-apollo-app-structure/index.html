Project Apollo: App Structure

March 5, 2024 · Ben Sheppard

In the first blog post of the Apollo project, I suggested that I would be using Java for the project as it’s a familiar language to me, and as I would like to deepen my knowledge of Java, it feels like a good candidate. I am aware that there are likely more lightweight/faster/efficient languages/frameworks/tools to use but this time around at least its Java!

I’ve been thinking about all the possible things that I want to do with this project. However, I am going to embrace an approach where I just get something working, so I can solve the core problem and then follow up with supporting structures. This means my first target is:

Create a service that takes an audio file and converts it to a structure that can be analysed for comparison purposes.

Step one: Create the project#

I’m using a Maven project structure here with the following POM file:

 3    <modelVersion>4.0.0</modelVersion>

 5    <groupId>dev.bensheppard</groupId>
 6    <artifactId>project-apollo</artifactId>

22    </dependencies>
23</project>
